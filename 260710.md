오늘 배운 코드 
Q1. requests.get() 의 역할?
웹사이트에 요청을 보내서 HTML 문서를 가져올수 있는 역할을 한다.
Q2. response.status_code == 200 을 확인하는 이유는?
웹페이지가 정상적으로 열렸는지 확인할수 있게 위해서다. 200이면 정상이고 다른 숫자들은 오류를 의미하는것이다.
Q3. soup.seLect(".prod-title") 의 의미는?
prod-title를 가진 모든 상품 제목을 리스트로 가져올수 있다.
Q4. for을 사용하는 이유는?
검색 결과의 모든 상품을 하나씩 확인하기 위해 사용한다.
Q5. if "복숭아" in i. text:는 무엇을 하는 코드?
상품 서치에 "복숭아"라는 단어가 포함되어 있는지 확인하는 조건문이다.
Q6. 수박이 아닌 검색 결과는 어떻게 구하는가?
전체검색 결과 개수 - 수박이 포함된 검색 결과 개수를 사요ㅇ해서 계산한다.
